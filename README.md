# JOY 長照服務品質管理系統（Prototype）

本專案為「長照服務品質管理」雛形前端，風格參考使用者提供的圖一（查詢）與圖二（表單）。

## 快速啟動
1. 直接用瀏覽器開啟 `index.html`（最簡單）
2. 或使用 VS Code Live Server（建議，避免瀏覽器對 fetch 的限制）
   - 安裝 Live Server 外掛
   - 右鍵 `index.html` → Open with Live Server

## 介面調整（2026-02-24 ~ v0.3）
- Topbar 固定顯示：新竹縣智慧福利服務躍升
- 左側主類別：長照服務品質管理（可展開）
  - 個案管理
  - 統計報表（預留）
- 查詢條件：新增「訪視人員」下拉、調整為 4 條排列
- 列表：新增「訪視人員」欄位

## 檔案結構
- `index.html`：查詢頁（條件 + 結果表）
- `form.html`：管理表單頁（含「訪視紀錄」按鈕）
- `assets/style.css`：主題與版面樣式（紫/粉色系）
- `assets/app.js`：共用 JS（載入 mock、下拉選單、工具函式）
- `assets/data.json`：mock 資料（含新竹縣鄉鎮下拉清單）
